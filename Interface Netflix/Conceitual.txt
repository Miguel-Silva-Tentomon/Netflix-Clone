# WRAPPER = ENVELOPAR 
Wrapper não é totalmente idêntico a um container

Uma classe de container é gerada para gerar maior facilidade na manipulação dos elementos filhos.
Os "filhos de primeiro grau" herdam as características do elemento pai.
